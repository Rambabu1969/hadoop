1. Download Java SDK, JRE and Hadoop 4.2.4 from https://drive.google.com/drive/folders/11nMEChq2-kCsOsyW51Y4n11j-zAH9Be5?usp=sharing
2. Unzip Hadoop 3.2.4 Setup file
3. Extract Hadoop to C drive, C:\Hadoop
4. Install Java SDK and JRE
5. Create Environment Variables
6. Run Command Prompt as Administrator
7. Launch Hadoop Cluster
   > hdfs namenode -format
   > start-all.cmd
8. Open Browser and Launth below sites for
Name node web page: http://localhost:9870/dfshealth.html
Data node web page: http://localhost:9864/datanode.html
Yarn web page: http://localhost:8088/cluster
